Getting Started with LibrePOS

Welcome to LibrePOS! Before you start using the system, there are a few essential setup steps:

1) Set up the database
LibrePOS uses SQLite by default, so you can get up and running quickly with minimal configuration. If youâ€™d rather use a different database (PostgreSQL, MySQL, etc.), you can configure any database supported by Flask-SQLAlchemy by updating your database connection settings accordingly.

2) Create a .env file and add a SECRET_KEY
LibrePOS requires a SECRET_KEY for secure session handling and other cryptographic features. Create a .env file in your project and define a strong secret key there.

3) Mark initial setup as completed (disables this welcome page)
To load the full application instead of the welcome screen, set the following in your .env file: INITIAL_SETUP_COMPLETED=true

4) Run LibrePOS in production configuration
Once your database and environment variables are in place, initialize the app using the production configuration: app = create_app("production")

After completing these steps, LibrePOS should be ready to start and serve requests.
